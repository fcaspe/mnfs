Frequently Asked Questions

What are the system requirements to run the plugin?

For the best experience, your system should meet these minimums:

macOS

  • Processor: Apple M1 chip or newer.
  • Host: A 64-bit DAW supporting AU or VST3 formats (e.g., Logic Pro, Ableton Live).

Windows

  • Processor: Intel Core i7 (11th Gen) or equivalent AMD Ryzen processor (or newer).
  • Operating System: Windows 11 (64-bit). Windows 10 is supported but not guaranteed for future updates.
  • Host: A 64-bit DAW supporting VST3 format (e.g., Ableton Live, FL Studio, Cubase).
How do I install the plugin?

We provide installers for both macOS and Windows. The plugin is installed automatically at:

macOS Locations:

  • AU: /Library/Audio/Plug-Ins/Components/
  • VST3: /Library/Audio/Plug-Ins/VST3/

Windows Location:

  • VST3: C:\Program Files\Common Files\VST3\

Have your DAW rescan for new plugins. It should appear in your effects list.

How do I get the lowest latency (10 ms or less) ?
  1. Adjust Your Buffer Size to 128 samples or less. This is the only critical setting that ensures you can comfortably play live with your instrument.
  2. (Windows Users) Use an ASIO Driver. For Windows, using an ASIO driver with an external audio interface is important for live use.
  3. Take a look at your project's sample rate. For now, only 44.1 kHz and 48 kHz sample rates are supported (to be extended soon!).

How can I report a bug or suggest an idea?

We rely on feedback from users like you to make this plugin great! Please send a detailed report to f.s.caspe@qmul.ac.uk.

To help us solve issues quickly, please include:

  • A clear description of the bug or your suggestion.
  • Your OS and DAW (e.g., macOS Sonoma 14.5, Ableton Live 12).
  • Steps to reproduce the issue you found.
